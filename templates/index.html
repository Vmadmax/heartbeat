Heartbeat

Was ist Heartbeat?
Heartbeat ist eine Software, die Gesichtserkennung auf eine große Anzahl an Bildern anwenden kann. Sie wurde hauptsächlich dazu entwickelt um zu zeigen, wie einfach es ist eine Massen-Gesichtserkennungssoftware zu bauen.

Wie funktioniert Heartbeat?
Heartbeat crawlt automatisch das Internet und lädt Bilder von Plattformen wie Instagram herunter. Auf heruntergeladenen Bildern wird dann eine Gesichtserkennung ausgeführt. Die erkannten Gesichter werden in einer Datenbank gespeichert um später mit einem hochgeladenen Bild abgeglichen zu werden.
